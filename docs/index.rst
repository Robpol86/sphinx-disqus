=============
Sphinx Disqus
=============

Embed `Disqus <https://disqus.com>`_ comments in Sphinx documents/pages.

``sphinx-disqus`` provides a ``disqus`` directive to be used in documents. It automatically adds Javascript needed by Disqus.
A working example is shown at the bottom of this page.
